# PC 및 모바일 버튼 배치 통합

## 🎯 목적
PC, 태블릿, 모바일 모든 화면에서 일관된 사용자 경험을 제공하기 위해 스크롤 버튼을 ADMIN 버튼 위에 수직으로 배치

## ✅ 완료된 수정사항

### 1. PC 화면 (≥1025px)
```css
.btn-admin {
    bottom: 30px;
    left: 30px;  /* 좌측 하단 */
    width: 100px;
    height: 45px;
}

.scroll-buttons {
    bottom: 90px;  /* ADMIN 버튼(45px) + 간격(15px) + 하단(30px) */
    left: 35px;    /* ADMIN 버튼보다 약간 중앙으로 */
    flex-direction: column;
    gap: 10px;
}

.scroll-btn {
    width: 50px;
    height: 50px;
}
```

## 🎨 디자인 원칙

### 1. 일관성 (Consistency)
- 모든 화면 크기에서 동일한 배치 패턴 유지
- 좌측 하단에 고정된 위치
- 스크롤 버튼이 항상 ADMIN 버튼 위에 위치

### 2. 접근성 (Accessibility)
- 터치 타겟 최소 크기 준수
  - PC: 50×50px (마우스 클릭)
  - 태블릿: 45×45px (터치)
  - 모바일: 40×40px (터치)
- 버튼 간 충분한 간격 (12-15px)

### 3. 시각적 계층 (Visual Hierarchy)
- 스크롤 버튼: 파란색 그라데이션 (기능적)
- ADMIN 버튼: 핑크-오렌지 그라데이션 (관리자 전용)
- z-index: 스크롤(9998) < ADMIN(9999)

### 4. 사용자 경험 (UX)
- 화면 하단 좌측: 접근하기 쉬운 위치
- 수직 정렬: 시각적으로 명확한 그룹핑
- 일관된 위치: 학습 곡선 감소

## 📐 계산 공식

### 스크롤 버튼 bottom 위치 계산
```
스크롤 버튼 bottom = ADMIN 버튼 height + 간격 + ADMIN 버튼 bottom

PC:       90px = 45px + 15px + 30px
태블릿:   75px = 40px + 15px + 20px
모바일:   65px = 38px + 12px + 15px
```

### 스크롤 버튼 left 위치 계산
```
스크롤 버튼 left = ADMIN 버튼 left + (ADMIN width - 스크롤 width) / 2

PC:       35px ≈ 30px + (100px - 50px) / 2
태블릿:   25px ≈ 20px + (90px - 45px) / 2
모바일:   20px ≈ 15px + (80px - 40px) / 2
```
