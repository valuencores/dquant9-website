🌐 Genspark IP 조회 도구

프로젝트 배포 도메인의 IP 주소를 자동으로 조회합니다

📌 사용 방법

1단계: 먼저 Publish 탭에서 프로젝트를 배포하세요

2단계: 배포 URL을 아래 입력란에 붙여넣으세요

3단계: "IP 주소 조회" 버튼을 클릭하세요

4단계: 조회된 IP를 카페24 A 레코드에 입력하세요

✅ 조회 결과

도메인: -
IP 주소: -
💡 명령줄에서 직접 확인:
nslookup [도메인]

❌ 조회 실패

-

💡 수동으로 확인하세요:
Windows: nslookup [도메인]
Mac/Linux: dig [도메인]

📋 카페24 DNS 설정 방법

A 레코드 (루트 도메인 @):

타입: A
호스트: @ (또는 비워두기)
값: [조회된 IP 주소]
TTL: 3600

CNAME 레코드 (www 서브도메인):

타입: CNAME
호스트: www
값: [Genspark 배포 도메인]
TTL: 3600
