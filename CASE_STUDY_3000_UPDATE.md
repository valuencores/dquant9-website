## 🧮 계산 공식 요약

### 단리 (Simple Interest)
```
최종 금액 = 원금 × (1 + 월이율 × 개월수)

예시 (3,000만원, 월 3%, 135일 = 4.45개월):
= 3,000만 × (1 + 0.03 × 4.45)
= 3,000만 × 1.1335
= 3,400.5만원 ≈ 3,401만원
```

### 복리 (Compound Interest)
```
최종 금액 = 원금 × (1 + 월이율)^개월수

예시 (3,000만원, 월 3%, 4.45개월):
= 3,000만 × (1.03)^4.45
= 3,000만 × 1.1414
= 3,424만원
```
