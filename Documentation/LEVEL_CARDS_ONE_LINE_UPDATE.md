## 🎨 CSS 변경

### 1. 그리드 레이아웃
```css
.level-stats-grid {
    display: grid;
    grid-template-columns: repeat(7, 1fr);  /* 7열 고정 */
    gap: 0.8rem;
}
```

### 2. 데스크톱 (1200px 이상)
```css
@media (min-width: 1200px) {
    .level-stats-grid {
        grid-template-columns: repeat(7, 1fr);  /* 7열 유지 */
        gap: 1.2rem;
    }
    
    .level-stat-card {
        padding: 1.8rem 1rem;
    }

        font-size: 2rem;  /* 숫자 크기 */
    }
    
    .level-stat-percent {
        font-size: 0.85rem;  /* 비율 크기 */
    }
}
```

### 3. 태블릿 (768px ~ 1199px)
```css
@media (max-width: 1199px) and (min-width: 768px) {
```


| 비율 | 0.7rem | 0.85rem |

| 화면 크기 | gap |
|-----------|-----|
| 모바일 | 0.8rem |

## 🎯 핵심 변경

1. **`grid-template-columns: repeat(7, 1fr)`**
   - 7개 박스를 동일한 너비로 배치

2. **gap 감소**
   - 1rem → 0.8rem (기본)
   - 1.5rem → 1.2rem (데스크톱)

3. **padding 감소**
   - 1.2rem → 1rem 0.8rem (기본)
   - 2.5rem → 1.8rem 1rem (데스크톱)

4. **폰트 크기 감소**
   - 숫자: 1.8rem → 1.5rem (기본)
   - 비율: 0.8rem → 0.7rem (기본)
