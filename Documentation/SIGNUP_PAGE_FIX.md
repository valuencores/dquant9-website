**작성일**: 2026-03-08  
**프로젝트**: D-QUANT 9.0  
**버전**: v6.2.5 → v6.2.6

---

## 📋 문제점

### **증상**
- 회원가입 메뉴를 클릭하면 로그인 확인 창이 먼저 표시됨
- 회원가입 페이지가 정상적으로 표시되지 않음

### **원인**
```html
<script src="js/check-login.js"></script>
```
- `check-login.js` 파일이 로드되어 모든 페이지에서 로그인 상태를 체크
- 회원가입 페이지에서도 로그인 모달이 표시됨

---

## ✅ 해결 방법

### 1. **check-login.js 제거**

**변경 전:**
```html
</script>
<script>document.addEventListener('DOMContentLoaded',function(){...});</script>
<script src="js/check-login.js"></script>

<!-- 로그인 모달 -->
<div id="loginModal">
    ...
</div>
```

**변경 후:**
```html
</script>
<script>document.addEventListener('DOMContentLoaded',function(){...});</script>

<!-- 푸터 -->
```

### 2. **로그인 모달 HTML 제거**

- 회원가입 페이지에는 로그인 모달이 필요 없음
- 약 100줄의 로그인 모달 코드 제거
