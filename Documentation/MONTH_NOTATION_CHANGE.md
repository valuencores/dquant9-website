## 🌍 국제 표준 준수

### "M" 단위 사용 이유
1. **국제 표준**: 
   - ISO 8601 기간 표기법
   - 금융권 국제 표준

2. **간결성**:
   - "6개월" → "6M" (3자 절약)
   - "12개월차" → "12M" (4자 절약)

3. **전문성**:
   - 금융/투자 업계 표준
   - 글로벌 스탠다드

4. **가독성**:
   - 숫자와 단위 명확히 구분
   - 테이블 컬럼 너비 절약
