## 📐 계산 방법

### **30% 감소 공식:**
```
새로운 크기 = 원래 크기 × 0.7
```

### **예시:**
- 3rem → 3 × 0.7 = 2.1rem
- 6rem → 6 × 0.7 = 4.2rem
- 2rem → 2 × 0.7 = 1.4rem
- 48px → 48 × 0.7 = 33.6px
