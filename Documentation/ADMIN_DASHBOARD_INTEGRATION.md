## 📋 개요

관리자 대시보드에 **최근 등록 회원 정보**를 통합하여, 관리자가 신규 회원 현황을 한눈에 파악할 수 있도록 개선했습니다.

---

## 🎯 구현된 기능

### 1. 최근 등록 회원 섹션 ✅

#### 위치
- 관리자 대시보드 상단 (통계 카드와 검색 바 사이)
- 토글 버튼으로 표시/숨기기 가능

#### 구성 요소
1. **통계 카드 (4개)**
   - 신규 등록: 6명
   - 투자금액: 2.9억원
   - 예상 월 배당금: 1,020만원
   - 최고 투자: 최은영 (1억원)

2. **회원 목록 테이블**
   - 9개 컬럼: No, 회원명, 이메일, 휴대전화, 투자자 레벨, 투자금액, 배당율, 예상 월 배당금, 가입일
   - 6명 회원 데이터 표시
   - 레벨별 색상 뱃지

3. **로그인 정보 박스**
   - 초기 비밀번호 표시: `DQ@2026`
   - 보안 안내 메시지

1. **로그인**
   - URL: `dquant/admin-login.html`
   - 이메일: `valuencores@gmail.com`

3. **최근 등록 회원 확인**
   - "최근 등록 회원 보기 (6명)" 버튼 클릭 (녹색, 펄스 효과)
   - 섹션이 슬라이드 다운으로 나타남
   - 통계 및 회원 목록 확인

4. **섹션 닫기**
   - 우측 상단 "닫기" 버튼 클릭
   - 섹션이 숨겨지고 버튼이 다시 나타남
