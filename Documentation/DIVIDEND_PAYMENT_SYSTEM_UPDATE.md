## 🔄 **다음 배당 예정일 계산 로직**

### **규칙:**

#### **1. 현재 날짜가 5일 이전인 경우:**
```
다음 배당일: 이번 달 5일
```

**예시:**
- 오늘: 2026년 3월 1일 → 다음 배당일: **2026년 3월 5일**
- 오늘: 2026년 3월 3일 → 다음 배당일: **2026년 3월 5일**

---

#### **2. 현재 날짜가 5일~24일인 경우:**
```
다음 배당일: 이번 달 25일
```

**예시:**
- 오늘: 2026년 3월 5일 → 다음 배당일: **2026년 3월 25일**
- 오늘: 2026년 3월 10일 → 다음 배당일: **2026년 3월 25일**
- 오늘: 2026년 3월 20일 → 다음 배당일: **2026년 3월 25일**

---

#### **3. 현재 날짜가 25일 이후인 경우:**
```
다음 배당일: 다음 달 5일
```

**예시:**
- 오늘: 2026년 3월 25일 → 다음 배당일: **2026년 4월 5일**
- 오늘: 2026년 3월 28일 → 다음 배당일: **2026년 4월 5일**
- 오늘: 2026년 3월 31일 → 다음 배당일: **2026년 4월 5일**

## 📊 **차월 계산 방법**

### **차월 = 가입일부터 다음 배당일까지의 개월 수**

**계산 공식:**
```javascript
차월 = (배당년도 - 가입년도) × 12 + (배당월 - 가입월) + 1
```

### **예시:**

#### **회원 A:**
- **가입일:** 2025년 11월 2일
- **현재 날짜:** 2026년 3월 8일 (오늘)
- **다음 배당일:** 2026년 3월 25일 (5일 지났으므로 25일)
- **차월 계산:**
  ```
  (2026 - 2025) × 12 + (3 - 11) + 1
  = 12 - 8 + 1
  = 5차월
  ```

#### **회원 B:**
- **가입일:** 2025년 12월 1일
- **현재 날짜:** 2026년 3월 8일 (오늘)
- **다음 배당일:** 2026년 3월 25일
- **차월 계산:**
  ```
  (2026 - 2025) × 12 + (3 - 12) + 1
  = 12 - 9 + 1
  = 4차월
  ```

## 💻 **코드 구현**

### **JavaScript 핵심 로직:**

```javascript
// 다음 배당일 계산 (매월 5일, 25일)
const today = new Date();
const currentDay = today.getDate();
let nextPaymentDate;

// 현재 날짜가 5일 이전이면 이번 달 5일
if (currentDay < 5) {
    nextPaymentDate = new Date(today.getFullYear(), today.getMonth(), 5);
}
// 현재 날짜가 5일~24일이면 이번 달 25일
else if (currentDay < 25) {
    nextPaymentDate = new Date(today.getFullYear(), today.getMonth(), 25);
}
// 현재 날짜가 25일 이후이면 다음 달 5일
else {
    nextPaymentDate = new Date(today.getFullYear(), today.getMonth() + 1, 5);
}

// 차월 계산 (가입일부터 다음 배당일까지의 개월 수)
const signupDate = new Date(member.signupDate);
const monthsDiff = (nextPaymentDate.getFullYear() - signupDate.getFullYear()) * 12 
                  + (nextPaymentDate.getMonth() - signupDate.getMonth()) + 1;
```
