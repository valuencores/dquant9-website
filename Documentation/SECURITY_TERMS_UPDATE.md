## 🎯 주요 추가 내용

### **1. 기밀 정보의 명확한 정의 (제2조)**
```
① D-Quant 9.0 알고리즘의 로직, 매매 전략, 파라미터
② 9단계 검증 프로세스 세부 내용
③ 플랫폼 소스 코드, 데이터 구조, API 정보
④ 고객 정보, 투자자 명단, 계약 조건
⑤ 미공개 경영 정보, 파트너십 계약
⑥ 타 이용자의 개인정보 및 자산 정보
```

### **2. 구체적인 보안 수칙 (제5조)**
```
• 비밀번호는 분기별 1회 이상 변경
• 공용 네트워크에서의 플랫폼 접속 자제
• 화면 잠금 및 자동 로그아웃 기능 활성화
```

### **3. 비밀유지 의무 기간**
```
서비스 이용 중 + 계약 종료 후 3년간
```

### **4. 보안 사고 보고 의무 (제14조)**
```
24시간 이내 즉시 보고:
① 계정 도용 의심
② 기밀 정보 유출 가능성
③ 비정상적 거래 또는 오류 발견
```

### **5. 손해배상 및 법적 조치 (제16조, 제17조)**
```
① 손해 전액 배상
② 영업상 손해, 명예 훼손, 법적 비용 포함
③ 금지 청구(가처분) 가능
④ 서비스 이용 중지 및 계약 해지
⑤ 중도 인출 패널티 적용 가능
```
