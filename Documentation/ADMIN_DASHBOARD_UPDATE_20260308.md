## 📅 업데이트 일자
**2026년 3월 8일**

## 📋 업데이트 개요

관리자 대시보드의 회원 관리 기능을 대폭 개선하여 개인정보 보호를 강화하고, 회원 정보 조회 기능을 향상시켰습니다.

---

## ✨ 주요 변경 사항

### 1. **회원 테이블 구조 개선**

#### 변경 전 (이전 버전)
| 가입일 | 회원명 | 투자금액 | 배당률 | 누적 차월 | 누적 배당금 | 투자자 레벨 | 관리 |
|--------|--------|----------|--------|-----------|-------------|-------------|------|
| 이메일과 전화번호가 노출 |

#### 변경 후 (현재 버전)
| 가입일 | 회원명 | 투자금액 | 배당률 | 누적차월수 | 누적배당금 | 투자자 레벨 | 관리 |
|--------|--------|----------|--------|------------|------------|-------------|------|
| 이메일/전화번호 제거 → 개인정보 보호 강화 |

**변경 이유:**
- ✅ **개인정보 보호 강화**: 이메일, 전화번호 등 민감한 개인정보는 메인 테이블에서 제거
- ✅ **가독성 향상**: 테이블이 더 간결해지고 핵심 정보에 집중
- ✅ **컬럼명 정확화**: "누적 차월" → "누적차월수"로 더 명확하게 표현

---

### 2. **회원명 클릭 → 상세 페이지 새 창 띄우기**

#### 기능 구현
```javascript
// 회원명 클릭 시 상세 페이지 새 창으로 띄우기
function viewMemberDetail(memberId) {
    const width = 1200;
    const height = 800;
    const left = (screen.width / 2) - (width / 2);
    const top = (screen.height / 2) - (height / 2);
    
    window.open(
        `member-detail.html?id=${memberId}`,
        '_blank',
        `width=${width},height=${height},left=${left},top=${top},scrollbars=yes,resizable=yes`
    );
}
```

#### 사용자 경험 (UX)
- 📱 **회원명을 클릭**하면 팝업 창으로 상세 정보 표시
- 🎯 **팝업 크기**: 1200x800px (화면 중앙 배치)
- 🔐 **개인정보 보호**: 이메일, 전화번호, 주소 등은 상세 페이지에서만 확인 가능
- 💼 **관리자 편의성**: 여러 회원 정보를 동시에 비교 가능 (멀티 탭/창)

4. 💵 **누적배당금**: 자동 계산된 총 배당금

#### 배당 지급 내역 테이블
- 📋 **지급일, 배당금, 차월, 상태, 비고**
- ✅ **자동 생성**: 투자 시작일부터 현재까지의 배당 내역

### 4. **누적배당금 자동 계산 로직**

#### 계산 공식
```
누적배당금 = 투자금액 × (배당율 ÷ 100) × 누적차월수
```

#### 예시
```javascript
// 회원: 최은영
// 투자금액: 100,000,000원 (1억원)
// 배당율: 4.0%
// 누적차월수: 9개월

월배당금 = 100,000,000 × 0.04 = 4,000,000원
누적배당금 = 4,000,000 × 9 = 36,000,000원
```

#### 실시간 계산
- ✅ 테이블 렌더링 시 자동 계산
- ✅ 통계 카드에서도 합산하여 표시
- ✅ 데이터베이스에 저장하지 않고 실시간 계산 (데이터 무결성 보장)

### 테이블 스타일

- 🌌 **다크 테마**: 고급스러운 그라데이션 배경
- 💠 **레벨 배지**: 레벨별 색상 구분 (Cosmos, Quantium, Platinum 등)
- 📊 **통계 카드**: 호버 효과, 아이콘 그라데이션
- 📋 **테이블**: 깔끔한 정렬, 호버 효과
