## 🎯 주요 추가 내용

### **1. 투자 단계별 수익 구조 (제3조)**

| 투자 단계 | 투자 금액 범위 | 월 고정 수익 | 연환산 수익률 |
|---------|------------|-----------|-----------|
| Trial (예비) | 1,000만 원 미만 | 2.0% | 24.0% |
| Core (본 A) | 1,000만 원 ~ 3,000만 원 | 2.5% | 30.0% |
| Growth (본 B) | 3,000만 원 ~ 5,000만 원 | 3.0% | 36.0% |
| Premium (본 C) | 5,000만 원 ~ 1억 원 | 4.0% | 48.0% |

### **2. 파트너 레퍼럴 수익 구조 (제8조)**

| 소개 투자자 단계 | 레퍼럴 지급율 | 연환산 레퍼럴 수익율 | 지급 성격 |
|--------------|------------|---------------|---------|
| Trial (예비) | 1.00% | 12.0% | 유치 보상금 |
| Core (본 A) | 1.20% | 14.4% | Passive Income |
| Growth (본 B) | 1.35% | 16.2% | Passive Income |
| Premium (본 C) | 1.50% | 18.0% | Passive Income |

### **3. 중도 인출 패널티 (제10조)**
```
운용 6개월 미만: 인출 원금의 10% 차감
운용 6개월 이상 ~ 1년 미만: 인출 원금의 5% 차감
```

### **4. 원금 보호 메커니즘 (제12조)**
```
• 9단계 검증 프로세스
• 3중 방어 체계(필터링, 헤징, 자동 실행)
• 평균 단가 대비 -5% 도달 시 스톱로스 자동 발동
```

### **5. 리스크 고지 (제15조)**
```
① 강한 원웨이 추세장에서 역추세 전략의 재진입 한도 소진 가능성
② 블랙 스완 사건으로 인한 예측 불가 손실 가능성
③ 거래소 API 장애로 인한 운용 지연 가능성
④ 불장에서 단순 보유 전략 대비 수익률이 낮을 수 있음
```

```

```

## 💰 수익 배분 (제7조)

### **월 고정 수익**
```
• 매월 말일 정산 후 지정 계좌로 지급
• 2026년 12월까지 단리형 구조
• 2027년 1월부터 복리 상품 선택 가능
```

### **IRT (Investment Reset Time)**
```
• Core 단계 이상: 매 6개월마다 원금 전액 송금
• 재진입 신청: 송금일로부터 15일 이내
```

### **복리 재투자 옵션**
```
• 최초 위임 시 또는 정산일 전 신청 가능
• 2027년 1월부터 4·6·8·12개월 거치형 선택 가능
```
