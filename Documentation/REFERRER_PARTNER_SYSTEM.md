## 📋 **주요 기능**

### **1. 회원명 옆에 추천인 표시**

#### **Before (이전):**
```
┌────────┬────────┬──────────┬────────┐
│ 가입일 │ 회원명 │ 투자금액 │ 배당률 │
├────────┼────────┼──────────┼────────┤
│2025.11.02│최은영 │100,000,000원│ 4.0% │
└────────┴────────┴──────────┴────────┘
```

#### **After (변경 후):**
```
┌────────┬──────────┬──────────┬────────┐
│ 가입일 │  회원명  │ 투자금액 │ 배당률 │
├────────┼──────────┼──────────┼────────┤
│2025.11.02│  최은영  │100,000,000원│ 4.0% │
│        │ (디퀀트) │          │        │
└────────┴──────────┴──────────┴────────┘
```

**표시 방식:**
```
회원명
(추천인)  ← 회색 작은 글씨
```

---

### **2. 관리자 대시보드 - 회원 추가/편집 폼**

#### **추가된 필드:**
```
┌─────────────────────────────────────┐
│ 추천인(파트너)                       │
├─────────────────────────────────────┤
│ [추천인 이름을 입력하세요 (예: 디퀀트)]│
├─────────────────────────────────────┤
│ 회원을 소개한 파트너 또는 추천인의   │
│ 이름을 입력하세요                   │
└─────────────────────────────────────┘
```

**위치:** 기본 정보 섹션 → 주소 입력란 다음

**특징:**
- ✅ 선택 입력 (필수 아님)
- ✅ 미입력 시 자동으로 "(디퀀트)" 표시
- ✅ 파트너 관리 용이

---

### **3. 회원가입 페이지**

#### **추가된 섹션:**
```
┌─────────────────────────────────────────┐
│ 추천인(파트너) (선택)                    │
├─────────────────────────────────────────┤
│ [디퀀트                            ]   │
├─────────────────────────────────────────┤
│ 회원님을 소개한 파트너 또는 추천인의     │
│ 이름을 입력하세요.                      │
│ 미입력 시 기본값 '디퀀트'로 설정됩니다.  │
└─────────────────────────────────────────┘
```

**위치:** 이메일 인증코드 입력란 다음, 약관 동의 섹션 이전

**특징:**
- ✅ 기본값: "디퀀트"
- ✅ 선택 입력 (필수 아님)
- ✅ 회원가입 시 자동 저장

## 🎯 **사용 시나리오**

### **시나리오 1: 관리자가 회원 추가**
1. 관리자 대시보드 → "회원 추가" 버튼 클릭
2. 기본 정보 입력 (이름, 이메일 등)
3. **"추천인(파트너)"** 필드에 추천인 이름 입력
   - 예: "김철수", "이영희", "박민수"
4. 투자 정보 입력
5. 저장
6. 회원 목록에 표시:
   ```
   김현수
   (김철수)
   ```

---

### **시나리오 2: 사용자가 회원가입**
1. 회원가입 페이지 접속
2. ID, 비밀번호, 이메일 입력
3. 이메일 인증
4. **"추천인(파트너)"** 필드:
   - 기본값: "디퀀트" (자동 입력됨)
   - 변경 가능: 실제 추천인 이름 입력
5. 약관 동의
6. 가입 완료
7. 회원 목록에 추천인 정보 포함되어 저장

---

### **시나리오 3: 추천인 미입력**
- **관리자 대시보드:** 추천인 필드 비워두고 저장 → 자동으로 "(디퀀트)" 표시
- **회원가입:** 기본값 "디퀀트"가 이미 입력되어 있음
