## 🧪 테스트 방법

### 1. 카카오톡 링크 캐시 초기화
카카오톡은 링크 정보를 캐싱하므로, 새로운 메타 태그를 적용하려면:
1. [카카오톡 링크 디버거](https://developers.kakao.com/tool/debugger/sharing) 접속
2. URL 입력: `https://dquant9.com/case-study.html`
3. "정보 보기" 클릭하여 썸네일 확인
4. 캐시 초기화 후 재확인

### 2. Facebook 디버거
1. [Facebook Sharing Debugger](https://developers.facebook.com/tools/debug/) 접속
2. URL 입력 후 "디버그" 클릭
3. 썸네일 이미지 및 메타 정보 확인
4. "다시 스크랩하기" 클릭하여 캐시 갱신

### 3. Twitter Card Validator
1. [Twitter Card Validator](https://cards-dev.twitter.com/validator) 접속
2. URL 입력 후 "Preview card" 클릭
3. Large Image Card 표시 확인

### 4. 실제 테스트
- **카카오톡**: 링크를 채팅방에 공유하여 썸네일 확인
- **Facebook**: 게시물 작성 시 링크 입력하여 미리보기 확인
- **네이버 밴드**: 링크 공유 시 썸네일 표시 확인


### 2. 용량 최적화
```bash

pngquant --quality=80-95 dquant-case-study-featured.png

# 또는 WebP 변환 (최신 브라우저 지원)
```
